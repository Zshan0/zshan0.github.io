We are going to use React for the frontend.
Setting a proxy
Add this to your package.json file:
"proxy": "http://localhost:3001"
JSON
React Router
Installation
npm install react-router-dom
Bash
Importing
import {BrowserRouter as Router, Switch, Route} from 'react-router-dom';
JavaScript
Wrapping
Whatever needs the routing functionality, we wrap it up as:
<Router> ...Routing stuff </Router>
JavaScript
Creating a Route
Rather than rendering everything in 1 place, we want things to render at different URLs. So for that we create a route and pass the component and the URL for which is should render.
<Router> <Route path = '/link' component = {Component} /> </Router>
JavaScript
Note: the route does prefix matching and renders everything that fits into its prefix.
Preventing multiple rendering
For this we use Switch, it will find the first match between the URL and the links given and render that out.
<Switch> ...Routes </Switch>
JavaScript
Exact Rendering
To make sure that the only time the rendering happens when the url matches exactly to the link we can use exact inside the Route.
<Route path = '/link' exact component = {}/>
JavaScript
Linking using Buttons
<Button href = "/link" />
JavaScript
