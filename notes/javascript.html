Garbage Collection
The principle followed to clean memory is reachability
All the objects that can be accessed directly or indirectly from <global> at a given instance are not deleted, if there exists an instance when the object becomes unreachable, the data is cleaned.
obj = { name: "batman" }; // The object is reachable here obj = null; // The object is no longer reachable, it is cleared from memory.
JavaScript
That was a simple example, but a more complicated example would be:
function assignRole(hero, sidekick) { hero.Side = sidekick; sidekick.Main = hero; return ({ mainGuy: hero, sacrifice: sidekick }) } let Duo = assignRole({ name: "batman" }, sidekick = { name: "Robin" }); // all the objects are in access, indirectly through Duo Duo = null; // now both hero and sidekick are inaccessible, hence they are cleaned
JavaScript
Cleaning Algorithm
