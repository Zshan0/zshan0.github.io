Normal operators
An operator is said to be normal if:
A†A=AA†A^{\dagger}A = AA^{\dagger}A†A=AA†
An operator which is Hermitian is by default also Normal.
A normal matrix is only Hermitian if and only if it has real eigenvalues:
Proof
Statement: A normal matrix with real eigenvalues is hermitian.
Since a normal matrix has to be square, and all square matrices have eigenvector decomposition in the complex field, we can write AAA﻿ and A†A^{\dagger}A†﻿ as:
A=XVX†A†=XV†X†A = XVX^{\dagger} \\ A^{\dagger} = XV^{\dagger}X^{\dagger}A=XVX†A†=XV†X†
Since the eigenvalues are real:
V=V†A=A†V = V^{\dagger} \\ A = A^{\dagger}V=V†A=A†
Proof by contradiction can prove the contrapositive statement too.
Spectral Decomposition
Statement
For a normal operator AAA﻿ over VVV﻿, there exists an orthonormal basis such that AAA﻿ is diagonalizable.
Rephrasing it, There exists a basis for VVV﻿ such that AAA﻿ is diagonalizable if and only if AAA﻿ is normal.
This can be split into 2 parts:
•
AAA﻿ is normal if AAA﻿ is diagonalizable
•
A is diagonalizable if AAA﻿ is normal
The 1st part is trivial since A=ATA = A^{T}A=AT﻿
Proof
We start with the statement that AAA﻿ is normal. which means:
A†A=AA†A^{\dagger} A = A A^{\dagger}A†A=AA†
The proof will be by induction.
Base case
For dim(V)=1dim(V) = 1dim(V)=1﻿, all the matrices are by default diagonal. So it satisfies the condition.
Induction case
Let λ\lambdaλ﻿ be an eigenvalue for AAA﻿. The eigenspace for the same would be Λ\LambdaΛ﻿.
A matrix PPP﻿ is a projector onto Λ\LambdaΛ﻿. and Q=I−PQ = I - PQ=I−P﻿

AP=λP and P2=Phence PAP=λPAP = \lambda P \text{ and }P^2 = P \\ \text{hence } PAP = \lambda PAP=λP and P2=Phence PAP=λP

This proves the existence of spectral decomposition for normal matrices.
