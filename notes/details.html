Object
A class to handle all the objects in the game that is the paddle, bricks, powerups and the walls.
Takes care of the movement and the collision if there are any collisions then it checks and readjusts the values.
Values
•
position: (x, y) of the top left pixel
•
dimensions: (width, height) of the object
•
color: color of the object
•
velocity: (vx, vy) velocity which is in float
Functions
•
Create: Spawns the object at the passed values (probably __init__)
•
Delete: call before deleting, replace the pixels with background
•
Displace: replace the original position with background and then put the color on the new location
•
Collision: based on the object that is colliding it with, will trigger the function. To be polymorphed for different subclasses of the Object class (such as Wall). The argument will also contain the object with which it is going to collide.
Player
Describes all the things about the player playing it.
Values
•
is_running: Checks if the game is running.
•
Paddle: Object which is the paddle
•
List of Balls: Represents the balls that haven't been dropped below, when 0 the player loses a life
•
Score: (need a formula)
•
Lives: Starts with 3, game stops when 0
Functions
•
init: to get create a paddle and a single ball, and initialize all the other values
•
Update score
•
Drop lives
Paddle
Subclass of Object.
Extra Values
•
Holding_ball: The object of the ball that is being held on the paddle if any.
Functions
•
Collision_result: Function which deals with what happens to the paddle when the object collides with the paddle (nothing happens to the paddle)
•
Movement function: Deals with input and moves the paddle
